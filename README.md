# Tic-Tac-Toe

A simple and responsive Tic-Tac-Toe game built with JavaScript, HTML, and CSS.

## Features

- **Responsive Design**
- **Intuitive User Interface**
- **Easy and Fun to Play**

## Technologies Used

- **JavaScript (ES6+)**
- **HTML5**
- **CSS3**

## How to Play

1. Open `index.html` in your browser.
2. Fill out the form and click submit.
3. Click on a cell to place your mark (X or O).
4. First player to get three in a row wins!
